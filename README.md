## Mô tả  
- **Cơ sở dữ liệu**: MySQL  
- **Phía máy chủ**: Tạo dịch vụ SOAP để hiển thị dữ liệu:  
  - Lấy tất cả các quốc gia từ cơ sở dữ liệu.  
  - Lấy tên thành phố theo `country code`.  

- **Phía máy khách**: Tạo ứng dụng WinForm để hiển thị:  
  - Tên quốc gia.  
  - Thành phố theo `country code`.
